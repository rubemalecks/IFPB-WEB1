Criando links

Site do IFPB

Por padrão, ao clicar um link ele abre na mesma janela do site que o link foi acessado.

É possível especificar em que janela o site do destino do link vai abrir, por meio do atributo target. Os valores úteis para esses atributos são:

  • _blank: abre o link em uma nova janela.
  • _self: é o padrão, abre na mesma janela.
