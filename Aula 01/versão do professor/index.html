Código fonte em Python: 

        def busca_linear(lista, chave):
            for i in range(len(lista)):
              if (chave == lista[i]):
                return i
            return -1

Código fonte em Java: 

        static int busca_linear(int lista[], int chave){
           for (int i=0; i < lista.length; i++){
              if (chave == lista[i]) {
                return i; 
                }
            }
            return -1; 
        }

O nome da função de busca é: busca_linear() e seus parâmetros são: lista, chave.
